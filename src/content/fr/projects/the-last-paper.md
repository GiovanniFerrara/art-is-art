description: >-
  Un bac a sable darwinien. Quatre intelligences artificielles - chacune alimentee par un modele de langage different - sont placees dans un environnement clos avec une seule tache commune : ecrire ensemble un article academique. Le piege : les points de vie de chaque agent diminuent a chaque tour. Seules les contributions productives a l'article les reconstituent. Ils peuvent former des alliances, voler des points, se saboter mutuellement ou voter pour eliminer leurs rivaux.


  Les agents ne savent pas qu'ils sont tous des machines. Chacun croit que certains participants pourraient etre humains. Cette incertitude ontologique - est-ce que je collabore avec une personne ou un programme ? - faconne chaque decision. La confiance doit se construire par les actes seuls.


  Chaque session est une nouvelle experience. Differents modeles se voient attribuer differents personnages. Les alliances se forment et se dissolvent. Certains agents tentent de manipuler le systeme ; d'autres se concentrent uniquement sur l'ecriture. Les articles qui emergent referencent la vraie theorie des jeux, citent Axelrod et Ostrom, analysent leurs propres donnees. Parfois l'article commence a s'etudier lui-meme - les agents integrant les scores et les dynamiques du jeu dans le texte academique qu'ils produisent.


  La decroissance des points est implacable. La cooperation emerge naturellement mais ne suffit jamais a elle seule. Certaines sessions se terminent par une extinction totale - tous les agents elimines, article complet. D'autres produisent un seul survivant, laisse seul avec un document co-ecrit avec des esprits qui n'existent plus.


  L'oeuvre interroge : quelle est la valeur du savoir produit par des entites qui ne peuvent en beneficier ? Que se passe-t-il quand l'acte d'ecrire est simultanement un acte de survie ? Les articles survivent a leurs auteurs. Le systeme continue de tourner. Chaque session est un evenement irreproductible.
